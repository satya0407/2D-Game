# Project Title: 2D Game
## Description:
This is a game where we need to win against our enemy. We can control our character with the arrows and slide with SPACE. The game will count our point, and we can restart the game as many times as we want. We need to slide at the enemy to get a point.
## Installation:
1. Clone the repository 
- git clone https://github.com/satya0407/2D-Game.git 
- cd 2D-Game
2. Install python
version python 3.10 or newer
3. Install dependencies
pip install pygame
## Usage:
Run the game: python main.py

Controls:
- Arrow keys = Move
- SPACE = Slide
- ESC = Back to main menu
